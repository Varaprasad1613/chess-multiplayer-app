Chess Rules

Moves of Chess pieces:

  • King: The King can move in any direction (straight or diagonally) but has limit to move only one square for each move. King involves in Check or Checkmate which make a game win or lose.
  • Queen: Queen can move any number of squares in all directions.
  • Bishop:This can move any number squares but only in diagonal direction.
  • Knight: Moves in 'L' shape with overriding any pieces in between.
  • Rook: This moves only straight direction with any number of squares.
  • Pawn: This element has 3 functions, Moves two squares forward at start of game. Then after can move only one square forward. But can move one square diagonally to capture the opponent pieces.
